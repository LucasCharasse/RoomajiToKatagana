compiler : javac principale.java
exécuter : java principale <fichier a lire>.txt


DESCRIPTION
-----------------------------

    traduit un texte composé de Roomaji en Katagana et Hiragana.

    ENTREES
    -----------------------------
        Prend un fichier texte .txt dont le contenue est du texte présenté est sous forme de prononciation syllabique
        (Roomaji). https://www.wikiwand.com/en/Hiragana
        Vous trouverez un exemple d'entrées dans "test.txt".

    RESULTAT 
    -----------------------------
        Génère un fichier HTML ( <fichier a lire>.html ) ou les Roomajis sont converti en Hiragana et Katagana.
        La lecture est adapter pour aller de la droite à la gauche.
